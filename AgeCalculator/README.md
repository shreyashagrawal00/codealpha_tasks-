# ⏳ Premium Age Calculator

A high-performance, visually stunning web application that calculates your precise age and provides fascinating life statistics with a modern, glassmorphic aesthetic.

## ✨ Overview

This Age Calculator goes beyond simple subtraction. Built with **Vanilla JavaScript**, **HTML5**, and **CSS3**, it offers a premium user experience with smooth animations, real-time validation, and a sleek "Obsidian Flux" inspired design.

## 🚀 Key Features

- **Precision Calculation**: Accurate to the day, accounting for leap years and varying month lengths.
- **Dynamic Statistics**: Discover more than just your age:
  - 🎂 **Next Birthday**: Know exactly when your next celebration is.
  - 📅 **Days Remaining**: A countdown to your special day.
  - 🌍 **Total Days Lived**: Your journey in days.
  - 💓 **Heartbeat Estimate**: Approximate total heartbeats in your lifetime.
- **Glassmorphism UI**: A state-of-the-art interface with backdrop filters, glowing orbs, and vibrant gradients.
- **Micro-Animations**: Smooth number counting effects and hover transitions for an interactive feel.
- **Responsive Design**: Flawless experience across mobile, tablet, and desktop devices.
- **Robust Validation**: Real-time error handling for invalid dates and future inputs.

## 🛠️ Technology Stack

- **Core**: HTML5, CSS3, JavaScript (ES6+)
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts
- **Aesthetics**: Glassmorphism, CSS Variables, Keyframe Animations

## 🎨 Design Philosophy

The application follows a **Dark Mode First** approach, using a deep slate palette (`#0f172a`) contrasted with vibrant neon accents in purple, pink, and blue. The use of `backdrop-filter` creates a premium "glass" panel effect, while floating background orbs provide depth and motion.

## 📦 Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/age-calculator.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd AgeCalculator
   ```
3. **Open the application**:
   Simply open `index.html` in any modern web browser.
